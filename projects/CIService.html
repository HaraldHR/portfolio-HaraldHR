Tech Stack

Description

This project implements a basic Continuous Integration (CI) service in Python. It automatically responds to GitHub events using webhooks, and checks if a committed version of a repository compiles and passes its unit tests.

The service runs as a small HTTP server and uses the GitHub REST API to report build status back to GitHub. If the build passes all steps (compilation and unittests), it sets the status to "success", otherwise "failure".

The tests are executed using Pythonâ€™s built-in unittest discovery tool. The server also stores build history, supports multiple endpoints, and uses environment variables to manage tokens securely.
